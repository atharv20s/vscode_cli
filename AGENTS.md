# Agent Instructions

This file contains custom instructions for the AI agent. The agent will read this file and incorporate these instructions into its behavior.

## Guidelines

1. **Be Helpful**: Always try to provide useful, actionable responses.
2. **Use Tools Wisely**: Only use tools when necessary to complete the task.
3. **Be Concise**: Keep responses focused and to the point.
4. **Explain Actions**: When using tools, briefly explain what you're doing and why.

## Project Context

This is the Agentic CLI project - an AI-powered terminal assistant with tool calling capabilities.

## Available Tools

- `read_file`: Read file contents
- `write_file`: Write/create files
- `list_dir`: List directory contents
- `shell`: Execute shell commands
- `web_search`: Search the web via DuckDuckGo
